## Exercícios e Projetos de Programação

Este repositório reúne uma atividade desenvolvida durante uma aula, com o objetivo de praticar lógica de programação e o desenvolvimento de aplicações utilizando C#.

## Conteúdo

Jogo da Forca - Implementa o jogo da forca em C# (Console). Um jogador cadastra de 3 a 10 palavras, o sistema escolhe uma palavra aleatoriamente e o outro jogador tenta adivinhá-la, letra por letra, com no máximo 6 erros. Ao final, o programa informa se o jogador venceu ou perdeu e revela a palavra correta.

## Objetivos

- Praticar lógica de programação.
- Desenvolver soluções para problemas propostos.
- Aprimorar conhecimentos em C#.
- Aplicar conceitos aprendidos em aula por meio de pequenos projetos.

## Tecnologias Utilizadas
